📦 Prefab UI Package Structure (schema_version 2)

This project packages and compiles target-platform HTML/JS frontends inside a layout mapping the Google Prefab Package specification. Instead of distributing shared binary libraries (.so / .dll), it distributes fully operational platform-specific UI modules.

📂 Folder Structure

File / Folder Path Role
prefab-dashboard-ui/prefab.json Package metadata (defines dependencies & scheme v2)
modules/dashboard/module.json Module definition & entry assets declaration
modules/dashboard/assets/web.x64/ Target platform folder for full browser assets
modules/dashboard/assets/chrome.v3/ Target platform folder for Chrome extension assets
output/web_x64_dashboard.html Generated final artifact running in browser
